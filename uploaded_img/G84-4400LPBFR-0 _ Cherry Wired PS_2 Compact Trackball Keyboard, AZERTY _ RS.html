    Product Details

    Trackball keyboard


    Ultra-compact AZERTY keyboard with integrated Ø16 mm trackball.
    It has 84 keys + 2 keys for the trackball (supplied with the Logitech diskette driver).
    5 light-emitting diodes for the Num, Caps, Scroll, Pad and trackball functions.
    PS/2 compatible and PS/2 mouse port for the trackball section.
